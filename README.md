## 📖 项目简介

**QQ群聊AI氛围官**是一个创新的社交AI产品，旨在解决QQ群聊中的三大核心痛点：

- 🥶 **群聊冷场** - 70%的QQ群日活跃消息不到10条
- 😔 **新人融入难** - 新成员入群后无人搭理，流失率高
- 💔 **情绪无人关注** - 负面情绪消息80%无人回复

我们的解决方案：让AI从**被动响应**变为**主动关怀**，成为群里真正的"氛围担当"。

## ✨ 核心功能

### 1. 智能破冰引擎 🔥
- 新人入群自动欢迎
- 分析群文化，生成个性化破冰话题
- 群冷场时主动发起讨论

### 2. 情绪感知陪伴 💙
- 实时检测负面情绪
- 提供温暖关怀和实际帮助
- 智能推荐群内资源和相关人员

### 3. 社交记忆管家 🎉
- 记住成员生日和重要时刻
- 自动生成群周年纪念报告
- 创造仪式感，增强归属感

### 4. 冲突智能调解 🤝
- 检测群内潜在冲突
- 及时介入，促进理解
- 维护群聊和谐氛围

### 5. 资源智能推荐 📚
- 打通QQ频道、QQ空间等生态
- 场景化推荐学习资源
- 促进群成员互助成长

### 6. 数据化运营 📊
- 群活跃度分析
- 成员互动关系图谱
- 高光时刻自动记录

## 🛠️ 技术架构

```
┌─────────────────────────────────────────────────────┐
│              QQ群聊AI氛围官                            │
├─────────────────────────────────────────────────────┤
│  感知层  │  NLP情感分析 + 场景识别 + 用户画像          │
├─────────────────────────────────────────────────────┤
│  决策层  │  主动触发引擎 + 个性化策略生成              │
├─────────────────────────────────────────────────────┤
│  执行层  │  内容生成 + 多模态交互 + 生态联动           │
└─────────────────────────────────────────────────────┘
```

### 核心技术
- **NLP情感分析**：RoBERTa-large 中文情感分类模型
- **场景识别**：规则引擎 + 机器学习混合方案
- **内容生成**：基于混元大模型微调 + RAG检索增强
- **用户画像**：多维度数据分析 + 隐私保护

## 📱 使用场景

### 游戏群
AI分析群聊记录，发现大家在讨论《黑神话悟空》，主动抛出话题："你们都卡在哪个BOSS了？"

### 学习群
检测到群沉默45分钟，AI发起小调查："这周最让你头疼的科目是啥？"并推荐学习资源。

### 兴趣群
成员生日时，AI不仅发送祝福，还回忆TA在群里的成长历程，营造仪式感。

## 🎨 产品特色

### 主动但克制
- 每天主动发言不超过3次
- 避免打扰，自然融入

### 个性化定制
- 根据群文化调整话术
- 不是通用模板，而是真正理解群

### 有温度的AI
- 不只是工具，更是有情感连接的群成员
- 促进人与人的连接，而非替代

### 可控可退
- 群主可设置AI活跃度（高/中/低/关闭）
- 用户可屏蔽AI消息
