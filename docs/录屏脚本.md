## 🎯 录屏技巧

### 录制前准备
1. **关闭通知**：系统通知、微信、QQ等全部关闭
2. **清理桌面**：隐藏敏感文件和图标
3. **浏览器准备**：
   - 只打开一个标签页
   - 隐藏书签栏
   - 使用无痕模式（避免显示历史记录）
4. **练习3-5遍**：确保流畅，背熟台词

### 录制设置
- **分辨率**：1920x1080
- **帧率**：30fps
- **音频**：清晰，无杂音
- **麦克风**：距离嘴巴15-20cm

### Mac 录屏步骤（QuickTime）
```bash
1. 打开 QuickTime Player
2. 文件 → 新建屏幕录制
3. 点击录制按钮旁边的箭头
4. 选择麦克风
5. 选择"录制所选部分"（框选浏览器窗口）
6. 点击录制
7. 完成后点击停止，保存为 .mov 格式
```

### 录制后处理
```bash
# 转换为 MP4（如果是 .mov 格式）
ffmpeg -i 录屏.mov -c:v libx264 -crf 23 -c:a aac -b:a 128k 产品演示.mp4

# 压缩视频（如果文件太大）
ffmpeg -i 产品演示.mp4 -vcodec h264 -acodec aac -b:v 2000k 产品演示_压缩.mp4
```

## 🚨 常见问题

**Q: 录屏时卡顿怎么办？**
A: 关闭其他应用，降低录制质量，或分段录制后剪辑

**Q: 说错话怎么办？**
A: 暂停后重新录制，或用剪辑软件剪掉错误部分

**Q: 时间超过3分钟怎么办？**
A: 加快语速，或删减部分场景（保留场景1、2、3即可）

**Q: 声音太小怎么办？**
A: 用剪辑软件提升音量，或重新录制
